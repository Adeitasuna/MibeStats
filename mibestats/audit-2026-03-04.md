## 1. SECURITE

### CRITICAL ‚Äî Credentials dans `.env` committe

Le fichier `.env` contient en clair :
- **DATABASE_URL** avec mot de passe Supabase
- **DIRECT_URL** idem
- **ME_BEARER_TOKEN** (Magic Eden API)

**Action immediate** : Rotation des credentials Supabase + Magic Eden, s'assurer que `.env` est dans `.gitignore`.

### HIGH ‚Äî SQL Injection (`$executeRawUnsafe`)

3 scripts de build utilisent `$executeRawUnsafe` avec interpolation de strings :
- `scripts/backfill-transfer-counts.ts:81-84`
- `scripts/build-owners.ts:73-82`
- `scripts/sync-owners.ts:57-62`

Des adresses wallet ou token IDs provenant de la blockchain sont injectes directement dans le SQL sans validation. **Fix** : remplacer par `$executeRaw` avec `Prisma.sql` tagge.

### HIGH ‚Äî Input validation insuffisante sur API routes

- `/api/tokens/map/route.ts` utilise `Prisma.raw()` pour les noms de colonnes.
- `/api/tokens/timeline/route.ts` valide les filtres par simple `.slice(0, 100)` sans whitelist enum.

**Fix** : Ajouter validation Zod avec enum strictes pour chaque filtre.

## 3. ARCHITECTURE

### CRITICAL ‚Äî FK manquante Sales -> Token

`Sales.tokenId` n'a pas de contrainte foreign key vers `Token.tokenId` dans `prisma/schema.prisma`. Risque d'orphelins si des tokens sont supprimes.

### HIGH (5 findings)

| # | Finding | Impact | Fichier |
|---|---------|--------|---------|
| 1 | 13 queries Eden en parallele (N+1) | Performance | `app/eden/page.tsx:32-136` |
| 2 | Index manquants | Queries lentes | `prisma/schema.prisma` |
| 3 | `@tanstack/react-query` installe mais jamais utilise | Bundle bloat (+50KB) | `package.json` |
| 4 | Fetch sans cache-control | Donnees stales | `components/eden/EdenContent.tsx:256-268` |
| 5 | Web3 libs (wagmi+viem+rainbowkit ~150KB) chargees globalement | Bundle size | `components/Web3Providers.tsx` |

**Index manquants recommandes :**
```prisma
@@index([tokenId, priceBera(sort: Desc)])
@@index([sellerAddress, buyerAddress])  // pour bubblemap
@@index([priceBera])                    // pour range queries
@@index([soldAt(sort: Desc), tokenId])  // pour recent sales
```
